# Express.js Backend API

## About This Project

This project was developed during my DecodeLabs Full Stack Development Internship.

The purpose of this project was to understand the basics of backend development using Node.js and Express.js. It includes simple REST API routes that can be tested using Thunder Client or Postman.

This project helped me learn how a backend server works and how APIs are created.

## Features

- Express.js Server
- REST API
- GET Request
- POST Request
- JSON Response
- Middleware Support

## Technologies Used

- Node.js
- Express.js
- JavaScript

## How to Run

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
node server.js
```

The server will start on:

```
http://localhost:3000
```

## API Endpoints

### Home Route

```
GET /
```

### Users Route

```
GET /api/users
```

### Create User

```
POST /api/users
```

The APIs can be tested using:
